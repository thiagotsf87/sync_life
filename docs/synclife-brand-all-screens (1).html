Regra da Wordmark
Como o nome "SyncLife" se comporta
SyncLife
Fundo escuro (Dark Foco / Dark Jornada)
"Sync" = branco (#dff0ff) · "Life" = gradiente (emerald → blue)
SyncLife
Fundo claro (Light Foco / Light Jornada)
"Sync" = navy escuro (#0f172a) · "Life" = gradiente darker (emerald-600 → blue-600)
Resumo: O ícone (logo Infinity Flow) é sempre idêntico — nunca muda. A única adaptação por tema é a cor do texto "Sync" na wordmark: branco em fundos escuros, navy em fundos claros. O "Life" sempre usa gradiente emerald→blue (ajustando apenas a saturação para contraste).

⚡ Componentes React Necessários

Para implementar tudo isso, precisamos de apenas 2 componentes:

<SyncLifeIcon size={34} animated /> — O ícone SVG isolado. Props: size (ajusta stroke-width e partículas automaticamente), animated (default: true, false para footer/favicon). Mesmo SVG sempre, independente do tema.

<SyncLifeBrand size="md" /> — Ícone + wordmark juntos. Tamanhos pré-definidos: sm (28+16px, footer), md (32+18px, navbar/onboarding), lg (38+18px, auth), xl (88+28px, hero), splash (120+32px). A cor do "Sync" adapta ao tema (claro/escuro) via CSS variable.

Esses 2 componentes cobrem 100% dos 7 contextos de uso. Nenhum SVG duplicado, nenhuma lógica condicional de cor para o ícone — apenas tamanho e animação se adaptam.
